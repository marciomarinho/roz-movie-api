================================================================================
                          API ENDPOINTS SUMMARY
================================================================================

GET /health
  Returns: {"status": "ok"}
  Auth: Not required

GET /
  Returns: Welcome message with doc links
  Auth: Not required

GET /docs
  Auto-generated Swagger UI
  Auth: Not required

GET /redoc
  Auto-generated ReDoc
  Auth: Not required

GET /api/movies
  List movies with pagination and filters
  Query: page, page_size, title, genre, year
  Auth: Required if API_KEY env var is set

GET /api/movies/search
  Search movies by title query
  Query: q (required), page, page_size, genre, year
  Auth: Required if API_KEY env var is set

GET /api/movies/{movie_id}
  Get single movie by ID
  Path: movie_id (integer)
  Auth: Required if API_KEY env var is set

================================================================================
                          TROUBLESHOOTING
================================================================================

Issue: "Module 'app' not found"
Solution: Make sure you're in the project directory and venv is activated

Issue: "Port 8000 already in use"
Solution: Use different port: uvicorn app.main:app --port 8001

Issue: "CSV file not found"
Solution: Download MovieLens data and place at ./data/movies.csv

Issue: "401 Unauthorized"
Solution: If API_KEY is set, include header: -H "X-API-Key: your-key"

Issue: "Docker build fails"
Solution: Try: docker build --no-cache -t movielens-api .

For more troubleshooting, see README.md "Troubleshooting" section
